# Changelog

## 1.0.0 (2026-03-17)

### Features

- **Free port** — Detect and kill processes occupying a port (`portwiz <port>`)
- **Smart switch** — Find the next available port (`portwiz <port> --switch`)
- **Dev mode** — Free a port and start your app (`portwiz dev <port> -- <cmd>`)
- **Doctor mode** — Scan common dev ports and report status (`portwiz doctor`)
- **Force mode** — Skip confirmation prompts (`--force`)
- **Cross-platform** — Works on Windows, macOS, and Linux
